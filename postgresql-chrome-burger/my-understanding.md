# PostgreSQL เป็นหนึ่งใน Structured Query Language
<p>เอาไว้ใช้จัดการDatabases เพราะเหมือนผมจะเข้าใจว่าจุดประสงค์ของพวกQueryคือเอาไว้เป็น DBMS อะไรงี้ครับ<p>
<p>Syntax SQLก็แรกๆอาจจะงงแต่ถ้าใช้บ่อยๆก็ไม่ยากครับ
<hr> 
    SELECT * FROM Students;
<hr>
<p> SELECT ก็คือเลือก , * ก็คือ All, FROM ก็คือมากจาก Tableไหน ในที่นี้คือ Students
